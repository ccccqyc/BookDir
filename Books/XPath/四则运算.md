|  符号   |         规则          |
| :---: | :-----------------: |
|  `+`  |        执行加法。        |
|  `-`  |        执行减法。        |
| `div` | 根据 IEEE 754 执行浮点除法。 |
|  `*`  |        执行乘法。        |
