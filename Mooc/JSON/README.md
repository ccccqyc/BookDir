总结
（1）JSON Android SDK官方的库
（2）GSON 适用于服务端的开发
（3）Gson 比 JSON 强大

JSONObject缺点：

1、不支持日期类型，需改为String
2、不支持解析javabean的json

JSON库总结：
功能：映射Java Object与json格式数据
1.通过Annotation注解来声明
2.支持自定义属性名称
3.支持包含或排除属性
4.支持自定义接口自己完成解析/生成过程
